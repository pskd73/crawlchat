---
title: User rate limits on questions
date: 2026-04-09
type: changelog
tags: focus
---

Now you can apply rate limits on unique users for the questions they ask. You can set either **Hourly** or **Daily** or both limits. To start with, it is rolled out for web widget. The user gets notified if the limit is reached. You can configure the limits from [Settings](https://crawlchat.app/settings#user-rate-limit) page.
