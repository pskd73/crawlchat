# CrawlChat Skills

This directory contains skill guides for working on CrawlChat.

## Available Skills

| File | Purpose |
|------|---------|
| [setup-dev.md](./setup-dev.md) | Dev server setup, login flow, useful commands, troubleshooting |
| [setup-collection.md](./setup-collection.md) | Knowledge collection creation, common fixes for MongoDB/pgvector/embedding issues |

## Using Skills

When working on CrawlChat tasks, check these guides for:
- Starting/stopping the dev environment
- Setting up knowledge collections
- Debugging common issues
- Following best practices (browser size, screenshots, git branches)
